Comparison

Equality

Receives two arguments of any type and returns true if they are equal.

eq(a: T, b: T): bool

Inequality

Receives two arguments of any type and returns true if they are not equal.

neq(a: T, b: T): bool

Control

If-Else

Receives a condition and two expressions. The condition must evaluate to a boolean. Both expressions must evaluate to the same type. If the condition is true, it returns the first expression. Otherwise it returns the second expression.

if(condition: bool, exp1: T, exp2: T): T

Try-Catch

Receives two expressions, which must evaluate to the same type. It returns the first expression unless it evaluates to an error, in which case, it returns the second expression.

try(exp1: T, exp2: T): T

Error

Throw error

Receives an expression of any type. It returns an error wrapping that expression.

error(a: T)

Debugging

Debug

Receives an expression of any type. It writes that expression in the standard output and returns the same expression.

debug(a: T): T
